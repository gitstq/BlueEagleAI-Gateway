## 🚀 快速开始 | Quick Start

### 前置条件 | Prerequisites

1. 注册账号：访问 [ahg.codes](https://ahg.codes) 完成注册
2. 获取 API Key：在控制台创建你的专属 API Key
3. 将 `base_url` 设置为 `https://ahg.codes/v1`

### 1️⃣ cURL 示例

```bash
curl https://ahg.codes/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "gpt-4o",
    "messages": [
      {"role": "user", "content": "Hello! What can you do?"}
    ]
  }'
```

### 2️⃣ Python 示例

```python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://ahg.codes/v1"  # 🦅 蓝鹰AI网关端点
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "system", "content": "你是一个有用的AI助手。"},
        {"role": "user", "content": "用简短的话介绍一下蓝鹰AI网关"}
    ]
)

print(response.choices[0].message.content)
```

### 3️⃣ Node.js 示例

```javascript
const OpenAI = require('openai');

const client = new OpenAI({
  apiKey: 'YOUR_API_KEY',
  baseURL: 'https://ahg.codes/v1'  // 🦅 蓝鹰AI网关端点
});

async function main() {
  const response = await client.chat.completions.create({
    model: 'gpt-4o',
    messages: [
      { role: 'system', content: 'You are a helpful assistant.' },
      { role: 'user', content: 'Briefly introduce BlueEagle AI Gateway.' }
    ]
  });
  console.log(response.choices[0].message.content);
}

main();
```
